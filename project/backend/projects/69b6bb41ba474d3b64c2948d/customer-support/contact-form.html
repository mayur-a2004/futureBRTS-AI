**Contact Form Page** ======================= ### Overview This is a basic contact form page that allows users to submit their queries or feedback to the website administrator. The form data is stored in a MySQL database using Core PHP. ### Database Schema The database schema for the contact form is as follows: ```sql CREATE TABLE contact_form ( id INT PRIMARY KEY AUTO_INCREMENT, name VARCHAR(255) NOT NULL, email VARCHAR(255) NOT NULL, phone VARCHAR(20), message TEXT NOT NULL, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ); ``` ### HTML Code (contact-form.html) ```html Contact Form
