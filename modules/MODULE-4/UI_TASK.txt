TASK: Build Feature Composer UI (Module-4)

Goal:
Create the UI where user can customize the project before building code.

Screens / Components:

1) Feature Toggles Panel
   - Switch list:
     * Auth Login
     * Email / OTP Verification
     * Payment Integration
     * Admin Dashboard
     * Notification System
     * API Security Layer
   - Each row = Label + Toggle (ON/OFF)
   - On change -> send event "FEATURE_TOGGLE_UPDATED"

2) Database Table & Field Builder
   - Section: "Tables"
     * Button: "Add Table"
     * For each table:
        - Table name input
        - List of columns (name + datatype dropdown + PK checkbox + delete icon)
        - "Add Column" button
   - Supported datatypes: string, number, boolean, date, file, json
   - Optional relationships placeholder: dropdown (One-to-Many, Many-to-Many)

3) Theme Composer
   - Primary color picker
   - Secondary color picker
   - Font selector (dropdown)
   - Corner radius slider (0–24)
   - Light/Dark mode toggle
   - Live preview pane on right side

Output file: output_v4_UI.md
