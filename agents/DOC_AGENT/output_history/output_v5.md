# Module 5: Launch Documentation

## 1. User Manual
### Getting Started
1.  **Sign Up:** Create an account with email/password.
2.  **Select Role:** Choose Student, Job Seeker, or Startup.
3.  **Create Project:** Use the "Project Builder" to generate your first blueprint.

### How to Build
- Enter project name and domain.
- Select difficulty.
- Click "Generate" and wait for the magic!

## 2. API Documentation
- **Spec:** OpenAPI 3.0 (Swagger UI available at `/api/docs`).
- **Auth:** Bearer Token required for all non-public endpoints.
